# Баг-репорт: дублирование ранее написанных сообщений в чате

## ID дефекта
CHAT-001 


## Приоритет
Medium 

## Описание
Все ранее написанные сообщения в чате дублируются в истории чата при отправке нового сообщения.

## Шаги для воспроизведения
1. Зайти в игровой чат
2. Написать и отправить первое сообщение "Привет"
3. Написать и отправить второе сообщение "Как дела?"
4. Написать и отправить третье сообщение "Что нового?"
5. Проверить историю чата

## Ожидаемый результат
В истории чата отображаются только уникальные отправленные сообщения:
1. "Привет"
2. "Как дела?"
3. "Что нового?"

## Фактический результат
При отправке каждого нового сообщения все предыдущие сообщения дублируются в истории чата:
1. "Привет"
2. "Привет", "Как дела?"
3. "Привет", "Как дела?", "Что нового?", "Что нового?"
