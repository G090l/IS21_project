# Баг-репорт: Персонаж движется при вводе названия комнаты лобби

## ID дефекта
LOBBY-001

## Приоритет
Medium

## Шаги для воспроизведения
1. Зайти в игры
2. Начать ввод названия комнаты в соответствующем поле
4. Наблюдать за поведением персонажа в фоне во время ввода

## Ожидаемый результат
При вводе названия комнаты лобби:
- Персонаж в фоне должен оставаться неподвижным
- Все анимации персонажа должны воспроизводиться корректно
- Управление не должно влиять на персонажа в интерфейсе

## Фактический результат
Во время ввода названия комнаты лобби:
- Персонаж в фоне начинает двигаться
