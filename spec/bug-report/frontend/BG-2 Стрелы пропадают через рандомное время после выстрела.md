# Баг-репорт: Стрелы пропадают через рандомное время после выстрела

## ID дефекта  
BG-2

## Приоритет  
Medium

## Шаги для воспроизведения  
1. Запустить игру.  
2. Выбрать лук (клавиша 2).  
3. Произвести выстрел (ЛКМ).  
4. Наблюдать за стрелой в полёте.  
5. Повторить выстрелы несколько раз.

## Ожидаемый результат  
Стрелы остаются видимыми до попадания в цель или до конца полёта.

## Фактический результат  
Стрелы бесследно исчезают через случайное время после выстрела, не достигая цели.
