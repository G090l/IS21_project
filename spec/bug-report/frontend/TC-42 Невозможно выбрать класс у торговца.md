# Баг-репорт: Невозможно выбрать класс у торговца

## ID дефекта
TC-42

## Приоритет
High

## Шаги для воспроизведения
1. Взаимодействовать с NPC-торговцем, который предоставляет услугу смены/выбора класса.
2. В открывшемся интерфейсе попытаться выбрать любой доступный для игрока класс.
4. Нажать кнопку "Выбрать" для применения изменений.

## Ожидаемый результат
Система позволяет выбрать класс После нажатия кнопки "Выбрать" класс успешно меняется.

## Фактический результат
Невозможно выбрать класс. Кнопка "Выбрать" остается неактивной, что делает процесс смены класса невозможным.
