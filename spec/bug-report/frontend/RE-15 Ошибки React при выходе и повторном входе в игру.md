# Баг-репорт: Ошибки React при выходе и повторном входе в игру

## ID дефекта
RE-15

## Приоритет
High

## Шаги для воспроизведения
1. Находиться в лобби.
2. Выйти из игры или запустить игру.

## Ожидаемый результат
Игра загружается корректно без ошибок.

## Фактический результат
После входа или выхода в игру появляются ошибки React: "Uncaught runtime errors".
