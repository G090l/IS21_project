# Баг-репорт: Персонаж движется при вводе текста в чат во время геймплея

## ID дефекта
CHAT-001

## Приоритет
Medium

## Шаги для воспроизведения
1. Зайти в игру
2. Открыть чат
3. Начать ввод текста в поле для сообщения
4. Наблюдать за поведением персонажа во время ввода текста

## Ожидаемый результат
При открытом чате и активном поле ввода игрок должен иметь возможность печатать сообщения, при этом:
- Персонаж остаётся неподвижным
- Управление персонажем временно блокируется
- Игровой мир продолжает работать в фоновом режиме

## Фактический результат
Во время ввода текста в чат:
- Персонаж продолжает реагировать на клавиши управления (WASD или аналогичные)
