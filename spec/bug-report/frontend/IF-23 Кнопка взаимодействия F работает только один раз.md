# Баг-репорт: Кнопка взаимодействия F работает только один раз

## ID дефекта
IF-23

## Приоритет
Medium

## Шаги для воспроизведения
1. Войти в игру.
2. Подойти к интерактивному объекту (NPC, дверь).
3. Нажать клавишу F для взаимодействия с объектом.
4. После завершения взаимодействия снова подойти к тому же или другому интерактивному объекту.
5. Попытаться использовать клавишу F для взаимодействия.

## Ожидаемый результат
Клавиша F должна работать для всех последующих взаимодействий без ограничений.

## Фактический результат
Клавиша F срабатывает только один раз. Для последующих взаимодействий необходимо кликать мышкой на кнопку взаимодействия в интерфейсе.
