# Баг-репорт: Персонаж отображается в комнате у всех игроков после перезагрузки страницы

## ID дефекта  
BG-3

## Приоритет  
High

## Шаги для воспроизведения  
1. Зайти в многопользовательскую комнату.  
2. Дождаться появления других игроков.  
3. Перезагрузить страницу    
4. Наблюдать за отображением персонажа у других игроков.

## Ожидаемый результат  
После перезагрузки персонаж должен появиться только у самого игрока и синхронизироваться с сервером корректно.

## Фактический результат  
Персонаж отображается в комнате у всех игроков, создавая визуальный дубликат или фантомную модель.
