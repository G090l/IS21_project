# Баг-репорт: Чат обновляется только после отправки первого сообщения после входа

## ID дефекта  
BG-4

## Приоритет  
Medium

## Шаги для воспроизведения  
1. Зайти в игру.  
2. Перейти в раздел чата.  
3. Не отправляя сообщений, наблюдать за чатом.  
4. Попросить другого игрока написать сообщение в чат.  
5. Проверить, отображается ли сообщение.

## Ожидаемый результат  
Сообщения других игроков должны отображаться в чате сразу после входа в игру, без необходимости отправки своего сообщения.

## Фактический результат  
Чат остаётся пустым до тех пор, пока игрок не отправит своё первое сообщение, после чего начинают отображаться все предыдущие и новые сообщения.
