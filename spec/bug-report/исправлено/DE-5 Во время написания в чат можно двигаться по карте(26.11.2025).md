# Баг-репорт: Во время написания в чат можно двигаться по карте

## ID дефекта
DE-5

## Приоритет
Medium

## Шаги для воспроизведения
1. открыть окно чата
2. Начать вводить текст в поле сообщения
3. Использовать клавиши WASD при наборе текста
4. Наблюдать за поведением персонажа

## Ожидаемый результат
При активном поле ввода чата невозможно одновременно двигаться по карте

## Фактический результат
Во время написания в чат персонаж двигается по карте при нажатии WASD
