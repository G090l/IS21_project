# Баг-репорт: невозможность нажать на кнопку "начать игру"

## ID дефекта
BG-001

## Приоритет
Medium

## Шаги для воспроизведения
1. Создать комнату
2. Изменить разрешение экрана 
3. Попытаться нажать на кнопку "начать игру"

## Ожидаемый результат
Кнопку можно нажать, её не перекрывает текст или другие элементы

## Фактический результат
Невозможно нажать на кнопку "начать игру"
