# Баг-репорт: Чат не загружается до отправки первого сообщения

## ID дефекта
DE-3

## Приоритет
[Medium]

## Шаги для воспроизведения
1. Зайти в игру
2. Написать в чат
3. Зайти с другого аккаунта
4. Сообщений нет
5. Написать сообщение
6. Сообщения появляются

## Ожидаемый результат
Чат обновляется в реальном времени на всех аккаунтах

## Фактический результат
Чат не обновляется пока в него не напишешь
