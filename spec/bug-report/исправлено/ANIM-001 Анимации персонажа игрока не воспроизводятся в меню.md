# Баг-репорт: Анимации персонажа игрока не воспроизводятся в меню

## ID дефекта
ANIM-001

## Приоритет
Mid

## Шаги для воспроизведения
1. Запустить игру
2. Перейти в главное меню
3. Найти модель персонажа игрока в меню 
4. Наблюдать за моделью персонажа

## Ожидаемый результат
Персонаж должен отображать предусмотренные анимации в соответствии с дизайном

## Фактический результат
Модель персонажа статична, все анимации отсутствуют. Персонаж застывает в базовой позе без какого-либо движения
