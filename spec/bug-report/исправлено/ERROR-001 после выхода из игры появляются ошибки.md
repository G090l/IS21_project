ERROR-001

## Приоритет
Medium

## Шаги для воспроизведения
1. Запустить игру
2. Выйти из игры 
3. Проверить наличие багов

Корректный выход из игры без ошибок

## Фактический результат
После выхода из игры появляются ошибки
