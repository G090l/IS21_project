# Баг-репорт: Невозможность создать комнату на 1 устройствах

## ID дефекта
DE-2

## Приоритет
High

## Шаги для воспроизведения
1. Создать лобби.
2. Попытаться создать еще одно лобби с другого аккаунта на том же устройстве.

## Ожидаемый результат
Можно создать несколько комнат

## Фактический результат
Можно создать только 1 лобби, даже при использовании разных аккаунтов на одном устройстве
