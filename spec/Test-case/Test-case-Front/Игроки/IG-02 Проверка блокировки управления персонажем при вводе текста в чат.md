| **Идентификатор** | IG-02 |
| **Приоритет** | Средний |
| **Название тест-кейса** | Проверка блокировки управления персонажем при вводе текста в чат |
| **Указание на модуль тестирования** | Игроки |
| **Исходные данные** | Игровая сессия запущена, персонаж находится в игровом мире |
| **Шаги тест-кейса** | 1. Нажать клавишу открытия чата <br>2. Начать ввод текста в поле для сообщения<br>3. Во время ввода нажимать клавиши управления движением (W, A, S, D) |
| **Ожидаемый результат** | Во время ввода текста в чат персонаж остается неподвижным, управление движением заблокировано. После закрытия чата управление восстанавливается. |
