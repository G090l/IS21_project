| **Идентификатор** | LOBBY-007 |
| **Название** | Проверка выхода игрока из комнаты после перезагрузки страницы |
| **Приоритет** | Высокий |
| **Модуль** | Лобби / Сетевое взаимодействие |
| **Исходные данные** | 1. Игра запущена<br>2. Игрок находится в комнате лобби |
| **Шаги** | 1. Перезагрузить страницу <br>2. Вернуться в ту же комнату лобби с другого аккаунта/устройства<br>3. Проверить список игроков в комнате |
| **Ожидаемый результат** | 1. Игрок, перезагрузивший страницу, исчезает из списка игроков в комнате<br>2. Комната не содержит «фантомных» игроков<br>3. Остальные игроки видят актуальный список участников |
