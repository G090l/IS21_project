| **Название** | Проверка возможности запуска игры и переноса игроков в игровую сессию |
| **Приоритет** | Высокий |
| **Модуль** | Лобби / Матчмейкинг |

| **Ожидаемый результат** | 1. Все игроки из комнаты переносятся в игровую сессию<br>2. Игровая сессия запускается корректно<br>3. Игроки видят игровое поле и могут управлять персонажами |
