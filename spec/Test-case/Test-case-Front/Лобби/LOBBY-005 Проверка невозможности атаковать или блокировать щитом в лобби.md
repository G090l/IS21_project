| **Идентификатор** | LOBBY-005 |
| **Название** | Проверка невозможности атаковать или блокировать щитом в лобби |
| **Приоритет** | Средний |
| **Модуль** | Лобби / Боевая система |
| **Исходные данные** | 1. Игра запущена<br>2. Игрок находится в лобби |
| **Шаги** | 1. Нажать ЛКМ (атака)<br>2. Нажать ПКМ (блок щитом)<br>3. Проверить реакцию персонажа |
| **Ожидаемый результат** | 1. Персонаж не атакует в лобби<br>2. Персонаж не блокирует щитом в лобби<br>3. Действия не влияют на других игроков в лобби |
