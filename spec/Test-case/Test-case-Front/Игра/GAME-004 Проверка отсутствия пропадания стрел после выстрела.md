| **Идентификатор** | GAME-004 |
| **Название** | Проверка отсутствия пропадания стрел после выстрела |
| **Приоритет** | Высокий |
| **Модуль** | Геймплей / Оружие |
| **Исходные данные** | 1. Игра запущена<br>2. Игрок находится в игровой сессии<br>3. Лук выбран (клавиша 2) |
| **Шаги** | 1. Выстрелить из лука (ЛКМ)<br>2. Следить за стрелой до конца полёта<br>3. Повторить выстрелы 5–10 раз<br>4. Проверить, пропадают ли стрелы раньше времени |
| **Ожидаемый результат** | 1. Стрелы остаются видимыми на всём протяжении полёта<br>2. Стрелы исчезают только после попадания в цель или при достижении максимальной дистанции<br>3. Ни одна стрела не пропадает преждевременно |
