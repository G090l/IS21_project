| **Название тест-кейса** | Проверка анимации атаки |
| **Указание на модуль тестирования** | Модуль анимаций персонажа, боевая система |
| **Исходные данные** | 1. Игрок загружен в игру<br>2. Персонаж Анака находится в состоянии покоя<br>3. Управление персонажем доступно<br>4. Есть доступная цель для атаки |
| **Шахи тест-кейса** | 1. Нажать кнопку атаки (обычно LMB/левый щелчок)<br>2. Наблюдать за анимацией атаки<br>3. Нажать кнопку атаки 3 раза подряд<br>4. Проверить комбо атаки<br>5. Нажать кнопку атаки после полного завершения анимации |
| **Ожидаемый результат** | 1. Анимация атаки запускается мгновенно при нажатии<br>2. Анимация проигрывается полностью от первого до последнего кадра<br>3. Комбо атаки имеет плавные переходы между ударами<br>4. Анимация завершается корректно, возвращаясь в состояние покоя |
